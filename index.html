About

I am a software engineer working on AI/ML infrastructure at Oracle Cloud Infrastructure (OCI), where I focus on building scalable platforms for deploying and serving machine learning models and large language models in production environments. My work sits at the intersection of distributed systems, cloud infrastructure, and machine learning systems engineering.

At OCI, I work on infrastructure for model deployment and inference platforms, designing systems that support high-throughput and low-latency model serving for enterprise workloads. This includes work on streaming inference endpoints, networking layers for model services, and infrastructure components that enable large-scale model serving across cloud environments.

In addition to industry work, I am actively involved in research and open-source contributions in machine learning systems and trustworthy AI. My research interests include privacy-preserving NLP, secure model deployment, multilingual speech systems, and efficient inference architectures for large-scale AI models.

My interests include:

  • AI infrastructure and model deployment platforms
  • Streaming inference systems
  • Secure and privacy-preserving machine learning
  • Large-scale distributed ML systems

Publications

  • Adaptive Neural Guardrails for Large Language Models - 2026 IEEE International Conference - (3SCEA)
    This paper presents a comprehensive AI-driven framework for guardrailing prompt engineering in Large Language Models (LLMs). We propose four integrated algorithms that leverage machine learning techniques to automatically detect, prevent, and mitigate risks in both input prompts and generated outputs. Our approach includes semantic prompt analysis, adaptive learning mechanisms, multi-modal threat detection, and output safety classification that work together to provide end-to-end protection.
  • Leveraging Synthetic Multilingual Speech for ASR Adaptation – 4th IEEE International Conference - (IATMSI)
    Contemporary automatic speech recognition (ASR) architectures necessitate extensive paired audio-text corpora for optimal convergence, yet acquisition of such datasets remains costly and time-consuming, particularly for multilingual deployment scenarios. This investigation presents a rigorous empirical analysis of synthetic speech generation leveraging XTTS for cross-lingual ASR domain adaptation across English (EN), Spanish (ES), and French (FR).
  • AI-Driven Threat Detection in Cloud Environment
    Cloud computing is the backbone of modern digital infrastructure and due to its rapid growth it has become vulnerable to sophisticated cyber threats. Traditional security systems, for example, are often based on static rules and signature-based detection, which means that they are not sufficient to combat zero-day attacks and complex multi-stage intrusions. This study proposes an AI-driven threat detection framework design with the objective of enabling autonomous cloud security by combining the hybrid design of graph neural networks (GNN) and transformer-based attention mechanisms for accurate anomaly identification with reinforcement learning-based adaptive response strategies.

  2. MCP Redis
    • Redis MCP – LREM Command Support
      Added support for the LREM Redis list command to the Redis MCP server implementation. This contribution improves command coverage and compatibility with Redis clients by enabling list element removal operations through the MCP command interface.
    • Redis MCP – Hybrid Search Integration
      Implemented enhancements to the Redis MCP server to support hybrid search capabilities, combining vector similarity search with traditional filtering mechanisms. This update improves the flexibility of Redis-backed AI applications by enabling more advanced retrieval patterns commonly used in retrieval-augmented generation (RAG) and semantic search systems.
  3. Llama Index

      Contributed improvements to the LlamaIndex framework, an open-source platform used for building retrieval-augmented generation (RAG) applications. The change enhances integration capabilities within the LLM application ecosystem.
  4. LightSeek SMG Contribution
    Enhanced request validation and tool selection reliability by requiring MCP toolset entries to have corresponding server configurations, and by enforcing that explicit tool choices include a declared, non-empty tools list. This update also unifies tool-name matching with stricter rules that respect per-tool enablement and MCP overrides, ensuring invalid selections are caught early and fail validation.

  • Operationalizing Enterprise AI: Designing Secure, Scalable Model Deployment Platforms
    Presented at the International Conference on Information Technology and Artificial Intelligence (ITAI 2026).

    As enterprises adopt large language models and AI copilots, the real challenge is no longer model accuracy, it is infrastructure. Internal AI platforms must support secure multi-tenancy, low-latency streaming inference, governance controls, region-aware deployment, and operational resilience at scale.

Attendees will learn how to move from experimental AI deployments to production-grade internal platforms that scale reliably across teams and regions.
